# 连接到app！

现在，我们已经创建了一个简单的Docusaurus网站，我们将学习如何将其与我们的应用程序集成。

## 为什么要连接到应用程序？

在我们的应用程序中，我们可能有一些文档，但是我们不想将其与我们的应用程序混合在一起。我们希望将其与我们的应用程序分开，以便我们可以将其放在一个地方，并且可以轻松地将其与我们的应用程序分开。
